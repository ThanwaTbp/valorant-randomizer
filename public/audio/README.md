# BGM Audio File

วางไฟล์ BGM ของคุณที่ `bgm.mp3` ในโฟลเดอร์นี้

## แหล่งเพลง royalty-free แบบไม่มีเสียงร้อง (ฟรี + ปลอดภัยเรื่องลิขสิทธิ์)

แนะนำ search keyword "cyberpunk" / "synthwave" / "electronic" / "tactical" / "futuristic":

1. **Pixabay Music** — https://pixabay.com/music/
   - ฟรี + ใช้งานเชิงพาณิชย์ได้ ไม่ต้อง credit
   - search: "cyberpunk", "synthwave", "electronic"

2. **Free Music Archive** — https://freemusicarchive.org/
   - มี filter เลือก license CC0 / CC-BY ได้
   - คุณภาพสูง

3. **YouTube Audio Library** — https://www.youtube.com/audiolibrary
   - ฟรีจาก Google ไม่มีลิขสิทธิ์

4. **Incompetech (Kevin MacLeod)** — https://incompetech.com/
   - CC-BY ต้อง credit เจ้าของ

ดาวน์โหลดเป็น `.mp3` แล้ว rename เป็น `bgm.mp3` วางที่นี่

## ขนาดไฟล์ที่แนะนำ

- < 5MB
- 128-192 kbps พอ
- ความยาว 1-3 นาที (loop ได้เนียน)
